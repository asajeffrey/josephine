    pub cx_: *mut JSContext,

Convenience class for imitating a JS level for-of loop. Typical usage:

ForOfIterator it(cx);
if (!it.init(iterable))
  return false;
RootedValue val(cx);
while (true) {
  bool done;
  if (!it.next(&val, &done))
    return false;
  if (done)
    break;
  if (!DoStuff(cx, val))
    return false;
}

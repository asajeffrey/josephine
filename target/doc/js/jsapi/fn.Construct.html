Invoke a constructor. This is the C++ equivalent of rval = Reflect.construct(fun, args, newTarget).

JS::Construct() takes a newTarget argument that most callers don't need. Consider using the four-argument Construct signature instead. (But if you're implementing a subclass or a proxy handler's construct() method, this is the right function to call.)
